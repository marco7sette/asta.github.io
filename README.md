# Asta del Mobile

React SPA per la rete di showroom **Asta del Mobile** — 10 punti vendita in Piemonte e Liguria.

## Stack

- **React 18** + Vite 5
- Single-file component (`src/App.jsx`)
- Nessuna dipendenza esterna oltre a React

## Avvio rapido

```bash
npm install
npm run dev
```

L'app sarà disponibile su [http://localhost:3000](http://localhost:3000).

## Build produzione

```bash
npm run build
npm run preview
```

## Sezioni principali

| Sezione | Descrizione |
|---|---|
| Home | Hero, categorie, promo, brand |
| Catalogo Prodotti | Filtri per ambiente, categoria, fascia prezzo |
| Promozioni Bloccate | Offerte a tempo con countdown |
| Shop Online | Acquisto diretto con carrello |
| Il Tuo Arredatore | Profili consulenti per showroom |
| Il Genio Risponde | Chatbot AI integrato |
| I Nostri Progetti | Galleria progetti realizzati |
| Soluzioni Contract | Servizi per hospitality e uffici |
